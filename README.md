# Password Generator

Generate secure passwords with configurable character sets (uppercase, lowercase, numbers, and symbols). This tool is designed for fast creation and one-click copying so you can use strong passwords for logins, APIs, and other accounts.

## Features

- Generate passwords based on options such as:
  - Length
  - Uppercase / lowercase letters
  - Numbers
  - Special characters
- One-click copy
- Easy-to-use UI for quick password creation

## Tech Stack

- React + TypeScript (Vite)
- TailwindCSS

## How to Run

```bash
npm install
npm run dev
```

Open the app at:

- http://localhost:3000

## Build

npm run build
